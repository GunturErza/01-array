# Sistem Pengelolaan Nilai Mahasiswa

Repositori ini berisi solusi sederhana untuk pengelolaan nilai 10 mahasiswa menggunakan array.

# Struktur File
Soal_01.ipynb: Menangani input data, mencari nilai ekstrem, dan memvisualisasikan nilai tertinggi/terendah.
Soal_02.ipynb: Menangani input data, menghitung status kelulusan (>= 60), dan memvisualisasikan data kelulusan.

# Analisis Kompleksitas
Setiap operasi dilakukan dengan iterasi tunggal pada array, sehingga:

| Operasi | Kompleksitas Waktu |
| :--- | :--- |
| Input Data | $O(n)$ |
| Pencarian Max/Min | $O(n)$ |
| Perhitungan Kelulusan | $O(n)$ |

*Catatan: $n$ adalah jumlah mahasiswa (dalam kasus ini 10).*
